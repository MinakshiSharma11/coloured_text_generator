Matrix's Discord Colored Text Generator

About

This is a simple app that creates colored Discord messages using the ANSI color codes available on the Mantine UI.

To use this, write your text, select parts of it and assign colors to them, then copy it using the button below, and send in a Discord message.

Source Code

This app runs entirely in your browser and the source code is freely available on GitHub.
